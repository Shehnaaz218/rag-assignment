# rag-assignment
This repository contains my implementation of a Retrieval-Augmented Generation (RAG) system using Python and LangChain.
## Project Overview
The goal of this project is to build a system that retrieves relevant information from documents and uses a language model to generate accurate answers based on that context.
## Technologies Used
- Python
- LangChain
- OpenAI
- Chroma Vector Database
- BeautifulSoup
- Gradio
## Workflow
1. Load documents from web sources
2. Split documents into smaller chunks
3. Generate embeddings using OpenAI
4. Store embeddings in a vector database
5. Retrieve relevant documents based on user queries
6. Generate answers using an LLM
## File Included
- Assignment__XYZ_RAG_solution.ipynb – Contains the full implementation of the RAG pipeline.

## Purpose
This assignment demonstrates how Retrieval-Augmented Generation improves LLM responses by providing relevant contextual information from external documents.
